# 🎨 Erwan's Funky Portfolio

Bienvenue dans mon **portfolio interactif en une page**, inspiré de l’univers visuel de **Tyler, The Creator** 🌸🌿  
Ce projet met en avant mes compétences avec une esthétique **funky, pastel et rétro**, comme un site que Tyler aurait pu designer.

---

## 🖼️ Ambiance visuelle

Palette de couleurs :
- 🌿 Vert pistache
- 🩷 Rose pastel
- 🧈 Crème
- 🌤️ Bleu ciel

Style :
- Typo expressive (manuscrite, bold, rétro)
- Fond ciel dégradé avec nuages stylisés en mouvement
- Animations douces, formes arrondies
- Effet papier/carton pour les blocs et sections

---

## ⚙️ Technologies

- [React](https://react.dev/) (via [Vite](https://vitejs.dev/))
- [Tailwind CSS](https://tailwindcss.com/)
- Icônes : Lucide / Custom SVG
- Animation CSS (scroll / transitions)

---

## 🧩 Fonctionnalités

- ✅ Portfolio **scrollable** sur une seule page
- ✅ Composants React **réutilisables** et bien structurés
- ✅ Sections nettes et séparées avec design responsive
- ✅ Animations d’entrée + transitions douces

---

## 🧱 Sections

| 🧷 Section     | 📋 Détail |
|---------------|-----------|
| **About me**  | Texte court, créatif et chill |
| **Projects**  | Blocs colorés, effets au hover, liens vers projets |
